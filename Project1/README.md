# 多模态AI互动式教学智能体

> 基于Vue3的多模态AI教学智能体前端项目，为中国大学生服务外包创新创业大赛设计

## 项目简介

本项目实现了一个多模态AI互动式教学智能体的前端界面，支持教师通过对话方式与AI智能体交互，生成高质量的教学课件。项目采用模块化设计，包含对话设计、知识库管理和课件预览三大核心功能模块。

## 技术栈

- **框架**: Vue 3 (Composition API + script setup)
- **构建工具**: Vite 5
- **路由**: Vue Router 4
- **样式**: 原生 CSS (响应式设计)
- **语言**: JavaScript (ES6+)

## 功能模块

### 1. 对话页面 (ChatPage)

- **多模态输入**: 支持文本输入、语音输入、文件上传
- **意图标注**: 上传文件时弹出意图标注弹窗
- **聊天式交互**: 类似ChatGPT的对话界面
- **个人信息管理**: 头像、用户名、教学学科、年级、班级人数
- **历史课件**: 查看历史对话生成的课件
- **快捷回复**: 智能提问触发器，提供快捷回复按钮

### 2. 知识库管理 (KnowledgeBase)

- **文档列表**: 展示已上传的教学参考资料
- **文件上传**: 支持PDF、Word、PPT、图片等格式
- **向量化进度**: 实时显示文档处理状态
- **文档预览**: 点击查看文档内容
- **统计面板**: 文档总数、已向量化、处理中

### 3. 课件预览 (CoursewarePreview)

- **PPT课件预览**: 缩略图导航、逐页编辑、翻页功能
- **Word教案预览**: 大纲导航、富文本渲染
- **创意内容展示**: 动画创意、互动小游戏
- **模式切换**: 只读/编辑模式
- **可收起侧边栏**: 对话历史展示
- **下载功能**: PPTX、DOCX文件下载

## 快速开始

### 环境要求

- Node.js >= 16.0.0
- npm >= 8.0.0

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

项目将在 `http://localhost:5173` 启动（Vite默认端口）

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录

### 预览生产构建

```bash
npm run preview
```

## 开发说明

### 纯前端实现

本项目目前为**纯前端实现**，包含完整的界面和交互逻辑：
- 所有数据均为模拟数据
- 没有后端API调用
- 方便与后端团队对接

### 与后端对接建议

3. **知识库**: 对接RAG检索增强系统
